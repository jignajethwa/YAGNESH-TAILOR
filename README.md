# YAGNESH-TAILOR
Create a simple, clean, and mobile-friendly website for a tailoring boutique called "Yagnesh Tailor Made Boutique".

Business details:
Name: Yagnesh Tailor Made Boutique
Location: Malad East, Mumbai, India
Phone Number: [ADD PHONE NUMBER HERE]

About the business:
Yagnesh Tailor Made Boutique specializes in designing and stitching traditional and custom clothing. The boutique designs suits, dresses, lehengas, blouses, and other custom garments. The focus is on quality stitching, perfect fitting, and personalized tailoring services.

Target users:
1. Customers who want to know about the boutique
2. The tailor (owner) who wants to record customer measurements digitally

Create the following pages:

1. Home Page
Include:
- Boutique name at the top
- Short introduction about tailoring services
- Services section listing:
  • Suit stitching
  • Dress stitching
  • Lehenga stitching
  • Blouse stitching
  • Custom tailoring
- Location: Malad East, Mumbai
- Contact phone number
- Button: "Enter Customer Measurements"

Design style:
Elegant, traditional boutique style
Colors: maroon, gold, white
Use tailoring icons such as needle, thread, and measuring tape.

2. Customer Measurement Form Page
Create a simple form where the tailor can quickly enter customer details and measurements.

Customer details:
Name
Phone number
Address
Garment type (Suit, Dress, Lehenga, Blouse, Custom)

Measurement fields:
Chest
Shoulder
Sleeve length
Neck
Waist
Hip
Length
Armhole
Thigh
Knee
Bottom
Pant length

Other fields:
Delivery date
Special instructions

After submitting the form, generate a clear measurement receipt.

3. Measurement Receipt Page
Display:
Boutique Name: Yagnesh Tailor Made Boutique
Customer Name
Phone Number
Garment Type
All Measurements
Order Date
Delivery Date
Notes

Include buttons:
Print Receipt
Download Receipt as PDF

4. Owner Dashboard
Create a simple dashboard where the tailor can:
- Add new customer measurements
- View previous orders
- Search customer by name or phone number
- Update order status (Pending, Stitching, Ready, Delivered)

Design requirements:
Very simple interface
Large buttons
Easy for non-technical users
Mobile responsive so it works well on phones.
